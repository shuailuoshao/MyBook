# MyBook 书籍管理桌面应用

基于 Electron 开发的精美书籍记录桌面应用，支持记录阅读书籍、管理读书笔记、多维度评分系统和文件夹分类。

## 功能特点

### 书籍管理
- 添加、编辑、删除书籍
- 记录书名、作者、开始/结束阅读时间
- 设置阅读状态：未开始、阅读中、已读完
- 卡片式书籍展示，响应式布局

### 文件夹分类
- 创建自定义文件夹
- 通过右键菜单将书籍移动到不同文件夹
- 侧边栏快速切换文件夹查看

### 读书笔记
- 支持导入 .txt 和 .md 文件作为笔记
- 笔记管理：查看、编辑、删除
- 笔记内容搜索

### 多维度评分系统
- 多种评分配置模板（标准配置、小说评分、漫画评分等）
- 20+ 评分维度：情节架构、世界观、人物设计、象征与意象、主题表达、文笔文风等
- 实时计算总分和平均分
- 可视化评分雷达图

### 数据管理
- JSON/CSV 数据导出
- JSON/CSV 数据导入
- 本地数据持久化存储

### 阅读统计
- 阅读状态分布饼图
- 月度阅读数量折线图
- 阅读时长统计
- 标签分类统计

### 其他特性
- 深色/浅色主题切换
- 关键词搜索
- 多维度筛选（状态、日期、评分、标签）
- 多种排序方式（书名、日期、评分等）

## 项目结构

```
MyBook/
├── main.js                 # Electron 主进程
├── preload.js              # 预加载脚本（安全通信）
├── renderer.js             # 渲染进程主逻辑
├── index.html              # 主界面
├── styles.css             # 主样式
├── package.json            # 项目配置
├── data/
│   └── books.json          # 书籍数据存储
└── dist/
    └── win-unpacked/       # 打包后的可执行文件
        └── MyBook书籍管理.exe
```

## 快速开始

### 运行开发版本

```bash
# 安装依赖
npm install

# 启动应用
npm start
```

### 打包应用

```bash
npm run build
```

打包后的 EXE 文件位于 `dist/win-unpacked/` 目录。

## 使用说明

### 添加书籍
1. 点击"添加书籍"按钮
2. 填写书名、作者、阅读日期
3. 选择阅读状态
4. 点击保存

### 管理文件夹
1. 点击侧边栏的"新建文件夹"按钮
2. 输入文件夹名称并确认
3. 右键点击书籍卡片 → "移动到..." → 选择目标文件夹

### 导入笔记
1. 右键点击书籍卡片 → "笔记"
2. 点击"导入笔记"按钮
3. 选择 .txt 或 .md 文件

### 评分书籍
1. 右键点击书籍卡片 → "评分"
2. 选择评分配置模板
3. 为各维度打分（-1 到 1）
4. 查看实时总分和雷达图

### 导出数据
1. 点击右上角菜单 → "导出"
2. 选择导出格式（JSON/CSV）
3. 选择保存位置

## 主题切换

点击顶部工具栏的主题切换按钮，可在浅色/深色主题之间切换。

## 数据存储

应用数据存储在 `data/books.json` 文件中。建议定期备份。

## 技术栈

- **前端**：HTML5、CSS3、JavaScript (ES6+)
- **桌面框架**：Electron 40+
- **图表库**：ECharts、Chart.js
- **数据存储**：JSON 文件
