- 添加键盘快捷键支持
- 添加全局搜索功能

- 更新现有方法使用 Toast 通知

### 新增功能模块

1. **Toast 通知系统** (`showToast` 方法)
   - 支持四种类型：success, error, warning, info
   - 自动关闭和手动关闭
   - 动画效果

2. **主题管理系统** (`toggleTheme`, `initTheme` 方法)
   - 深色/浅色模式切换
   - 本地存储偏好
   - 图标自动更新

3. **键盘快捷键系统** (`handleKeydown` 方法)
   - 支持 Ctrl/Cmd 组合键
   - 防止在输入框中触发
   - 智能快捷键映射

4. **右键菜单系统** (`showContextMenu`, `handleContextMenu` 方法)
   - 智能位置计算
   - 事件委托
   - 与现有功能集成

5. **全局搜索系统** (`handleGlobalSearch` 方法)
   - 防抖处理
   - 多字段搜索
   - 实时反馈

## 使用说明

### 深色模式
- 点击 header 右侧的月亮/太阳图标切换主题
- 主题偏好会自动保存

### 键盘快捷键
- 在非输入区域使用快捷键
- 支持 Mac (Cmd) 和 Windows (Ctrl)

### 右键菜单
- 在书籍卡片上右键点击
- 选择相应操作

### 全局搜索
- 在 header 左侧搜索框输入关键词
- 支持实时搜索

### Toast 通知
- 所有重要操作都会有 Toast 提示
- 点击右上角的 × 可以手动关闭

## 性能优化

1. **CSS 变量**：减少重复样式，提高主题切换性能
2. **防抖处理**：搜索输入防抖，减少不必要的渲染
3. **事件委托**：右键菜单使用事件委托，减少内存占用
4. **本地存储**：偏好设置本地存储，减少网络请求
